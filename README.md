# 📺 IPTV Auto-Repair (GitHub Native)

这是一个基于 Python 的 IPTV 频道自动维护工具。它直接运行在 GitHub Actions 上，每天自动检测、搜索并修复存放在 `output/` 目录下的播放列表。



## 🚀 工作原理

1. **扫描**：自动识别 `output/` 下所有的 `.m3u` 文件。
2. **检测**：使用 `Validator` 模拟播放器进行深度测试（检查状态码、内容类型、数据流）。
3. **修复**：若链接失效，`Spider` 会在网络上精准搜索同名频道的新源。
4. **决策**：
   - 找到新源 -> 替换旧链接。
   - 未找到新源 -> **保留旧链接**（默认策略，防止网络波动导致频道永久丢失）。
5. **分发**：更新后的文件自动推送到仓库。

## 📂 目录说明
- `core/`: 核心逻辑组件 (Loader, Spider, Validator)。
- `output/`: 存放你的 M3U 文件，支持任意添加新文件。
- `main.py`: 自动化调度入口。

## 🛠️ 维护建议
- **增加文件**：直接将 M3U 文件拖入 `output/` 目录，命名请使用英文/拼音。
- **手动触发**：在 GitHub 仓库的 **Actions** 标签页，可以手动点击 `Run workflow` 立即开始修复。



---
*注：本项目运行在 GitHub Actions 环境中，IP 地址位于美国。若部分源有地域限制（如仅限日本或中国 IP），验证结果可能受影响。*
